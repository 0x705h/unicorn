Installation
============

1. Softwarerequirements
-----------------------

  Linux
  - ruby >= 1.9.3
  - rubygems
  - make
  - gcc

  Mac OS
  - ruby >= 1.9.3
  - rubygems
  - make
  - XCode

2. Install unicorn
------------------
  cd <path to unicorn>
  ./make.sh install

3. Install ruby binding
-----------------------
  cd bindings/ruby
  make install
